# Vision Skill

The Vision Skill enables the agent to "see" the page using visual screenshots and computer vision models.

## Capabilities
- **capture_for_ai()**: Takes a high-resolution screenshot and prepares it for LLM analysis.
- **analyze_visuals()**: Identifies issues like color contrast, focus indicators, and spacing that automated tools often miss.

## Use Cases
- Verifying focus state visibility.
- Checking contrast on complex gradients or images.
- Validating the logical visual flow of the page.
